# Data-Driven Product Recommendations for Bellabeat

## Live Project
[View Project Here (RPubs)](https://rpubs.com/LindsayWhipple/1079783)

## Overview
This project was completed as part of the Coursera Google Data Analytics Professional Certificatation program and focuses on analyzing customer data to identify patterns and segment users based on their interactions. The project serves as a foundational application of exploratory data analysis and segmentation techniques in a structured learning environment. 

## Objective
The goal of this project is to:
- Explore customer behavior and usage patterns
- Identify distinct customer segments based on engagement
- Analyze differences between customer groups
- Apply data analysis techniques learned through the Google Data Analytics curriculum

## Dataset
Source: [Coursera (course-provided dataset)](https://www.kaggle.com/datasets/corissajohnson/bellabeat-coursera-data-analytics-capstone/data)

Designed to support exploratory data analysis and clustering techniques.

Contains:
- Customer activity and engagement metrics
- Behavioral features used for segmentation

## Tools and Technologies
- R
- tidyverse (dplyr, ggplot2)

## Methodology
- Cleaned and prepared customer data for analysis
- Conducted exploratory data analysis to identify patterns
- Applied segmentation techniques to group customers
- Visualized differences between segments using charts and summaries

## Key Insights
- Customer groups exhibit distinct behavioral patterns based on engagement levels
- High-engagement users differ significantly from low-engagement users in activity metrics
- Segmentation helps identify potential target groups for marketing or product strategies
- Behavioral data provides valuable insights into user retention an interaction trends

- Distribution of Engagement Metrics
